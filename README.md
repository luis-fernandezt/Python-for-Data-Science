## üìö Glosario

| **T√©rmino / F√≥rmula**            | **Definici√≥n**                                                                 |
|----------------------------------|--------------------------------------------------------------------------------|
| `Media:` \( \mu = \frac{1}{n} \sum x_i \)     | Promedio aritm√©tico de un conjunto de datos.                               |
| `Varianza:` \( \sigma^2 = \frac{1}{n} \sum (x_i - \mu)^2 \) | Mide la dispersi√≥n de los datos.                    |
| `Desviaci√≥n est√°ndar:` \( \sigma = \sqrt{\sigma^2} \)      | Ra√≠z cuadrada de la varianza.                          |
| `Error est√°ndar:` \( SE = \frac{\sigma}{\sqrt{n}} \)       | Precisi√≥n de la media muestral.                         |
| `Z-score:` \( z = \frac{x - \mu}{\sigma} \)   | N√∫mero de desviaciones est√°ndar que un dato se aleja de la media.          |
| `IC 95%:` \( \bar{x} \pm 1.96 \cdot SE \)      | Intervalo de confianza para la media con 95% de seguridad.                 |
| `Regresi√≥n lineal:` \( y = \beta_0 + \beta_1 x + \epsilon \) | Relaci√≥n lineal entre variable independiente y dependiente.              |
| `Correlaci√≥n:` \( r = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum(x_i - \bar{x})^2 \sum(y_i - \bar{y})^2}} \) | Asociaci√≥n entre dos variables. |
| `linear regression:`              | \( Y = \beta_0 + \beta_1 X + \epsilon \) ‚Üí modelo de relaci√≥n lineal     |
| `R¬≤ (R-squared):`                 | Proporci√≥n de varianza explicada por el modelo                                |
| `central limit theorem:`          | La media muestral se distribuye normal si n es suficientemente grande         |
| `type I error (Œ±):`               | Rechazar la hip√≥tesis nula cuando es verdadera (falso positivo)               |
| `type II error (Œ≤):`              | No rechazar la hip√≥tesis nula cuando es falsa (falso negativo)                |
| `p-value:`                        | Probabilidad de obtener un resultado igual o m√°s extremo, dado H‚ÇÄ verdadera   |
| `one-tailed test:`                | Prueba estad√≠stica con hip√≥tesis direccional (mayor o menor)                  |
| `two-tailed test:`                | Prueba estad√≠stica donde interesa desviaci√≥n en ambas direcciones             |
| `hypothesis:`                     | Suposici√≥n que puede ser probada con datos                                    |
